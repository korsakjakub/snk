# snk
[![Go Report Card](https://goreportcard.com/badge/github.com/korsakjakub/snk)](https://goreportcard.com/report/github.com/korsakjakub/snk)

simple snake game written on a train

build simply with

```
go build -o build/
```

or run with

```
go run ./...
```
or test with

```
go test -v ./...
```
